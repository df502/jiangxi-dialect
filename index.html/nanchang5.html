三、称呼人和形容事物的南昌话

崽俚 / zǎi lǐ
小伙子、男孩。“那个崽俚长得蛮帅。”
娌子 / lǐ zi
姑娘、女孩。“他家娌子考上大学了。”
姆妈 / mǔ mā
妈妈。公交上听到小孩喊。
嗲嗲 / diā diā
爸爸。口语称呼，戴师傅说他小时候叫“嗲嗲”。
公公 / gōng gong
爷爷。
婆婆 / pó po
奶奶。
绝杀 / jué shā
太好了、绝了。“这碗拌粉绝杀！”
咋呼 / zà hu
大惊小怪。“莫咋呼，小事一桩。”
结根 / jié gēn
固执、爱钻牛角尖。“那个人好结根，说不通。”
打野 / dǎ yě
开小差、分心。“上课莫打野。”
撩撇 / liāo piē
简单、容易。“这事好撩撇，一下就弄好了。”
作死 / zuò sǐ
调皮、胡闹（有时是骂小孩）。
✍️ 戴师傅说“绝杀”这个词年轻人说得更多，老一辈不说。他还说有个词“冒泡”是出风头，但我不确定是不是南昌话。
