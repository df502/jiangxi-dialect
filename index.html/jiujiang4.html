三、九江方言日常用语

以下词汇主要来自九江市区老一辈的口语。九江话说“吃过”不说“恰饭”，有儿化音，与其他江西方言明显不同。

吃过 / chī guò
吃饭。例：“你吃过冇？”
么 / miē
什么。“你搞么？”（你干嘛？）
冇 / mào
没有。“冇得”就是没有。
晓得 / xiǎo de
知道。否定说“不晓得”。
困告 / kùn gào
睡觉。“早点困告，明天还要上班。”
过早 / guò zǎo
吃早饭。与武汉话相同。
话事 / huà sì
说话、聊天。“几个老人在门口话事。”
打野 / dǎ yě
开小差。“上课不要打野。”
掇 / duō
搬、端。“帮我把椅子掇过来。”
斫 / zhuó
砍。古汉语用字，“斫柴”。
靓 / liàng
漂亮。“这姑娘好靓。”
结根 / jié gēn
固执、爱抬杠。“那人好结根，讲不通。”
※ 注：拼音为近似发音，实际语音存在细微差异。部分词汇与赣语相通，部分受官话影响。
