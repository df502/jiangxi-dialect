三、南昌人日常怎么说话（一）

下面这些词是我从戴师傅、早餐店老板、公交上偷听来的。拼音是我自己瞎标的，不一定准。

恰饭 / qiá fàn
吃饭。早餐店老板娘每天早上喊：“进来恰饭不？”
嬲 / niǎo
玩、闲逛。戴师傅说“周末带孙子去公园嬲”。
莫作声 / mò zuō sēng
别说话。他接电话时喊“莫作声，听我讲”。
里手 / lǐ shǒu
内行、厉害。“他修车好里手，一会就修好了。”
冒得 / mào dé
没有。我问“有辣椒吗”，老板娘说“冒得，自己去加”。
克 / kè
去。“你克哪里？”（你去哪里？）
晓得 / xiǎo de
知道。“不晓得”就是不知道。
啥 / sá
什么。但戴师傅说年轻人不常说，说“么里”多。
箸子 / chù zi
筷子。“拿双箸子来。”这个词很古。
昼饭 / zhòu fàn
午饭。“吃过昼饭冇？”
夜饭 / yá fàn
晚饭。同上。
困觉 / kùn gào
睡觉。戴师傅说“早点困觉，明天考试”。
※ 本来还记了“瓦罐汤”、“拌粉”这些吃的东西，但发音跟普通话差不多，就不写了。
