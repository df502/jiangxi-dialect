三、饮食、称谓与生活词汇

油馓 / yóu sǎn
一种油炸面食，九江特色早点。
豆粑 / dòu bā
绿豆或大米磨浆摊成的饼，可炒可煮。
煨汤 / wēi tāng
瓦罐煨汤。九江人也爱喝。
老爹 / lǎo diē
爷爷或老年男子。
娭毑 / āi jiě
奶奶或老年妇女。湖南江西一带常用。
佬俵 / lǎo biào
江西人之间的亲切称呼，九江亦用。
崽 / zǎi
儿子。“我崽在南昌读书。”
女 / nǚ
女儿。
屋里 / wú lǐ
家里。“回屋里去。”
灶下 / zào hā
厨房。老派说法。
✍️ 九江方言词汇融合了码头文化、商业文化和农耕文化，许多词语生动描绘了九江人耿直、爽快的性格。
